# 📊 Sales Performance Dashboard (Power BI)

## 📌 Overview

This project is a **Sales Performance Dashboard** built using Power BI.
It provides insights into sales trends, profit, orders, and product performance across different regions and categories.

The goal of this dashboard is to help stakeholders quickly understand business performance and make data-driven decisions.

---

## 🚀 Key Features

* 📈 **Total Sales KPI** – Overall revenue performance
* 💰 **Profit KPI** – Total profit generated
* 📦 **Total Orders KPI** – Number of orders
* 📅 **Monthly Sales Trend** – Sales performance over time
* 🏷️ **Category Analysis** – Sales by product category
* 🌍 **Regional Analysis** – Sales distribution by region
* 🥇 **Top Products** – Top 5 products by sales
* 🎯 **Interactive Slicer** – Filter dashboard by region

---

## 🛠️ Tools & Technologies

* Power BI
* DAX (Data Analysis Expressions)
* Data Visualization Techniques

---

## 📂 Dataset

The dataset used in this project contains sales transaction data including:

* Order Date
* Sales
* Product Category & Name
* Region
* Customer details

*(Dataset sourced from Kaggle / sample sales dataset)*


---

## 📊 Key Insights

* Sales show a steady increase towards the end of the year
* Technology category generates the highest revenue
* West and East regions contribute the most to sales
* A small number of products contribute significantly to total sales
